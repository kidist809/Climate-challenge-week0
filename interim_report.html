1. Understanding and Defining the Business Objective

This project analyzes daily climate data from five African countries — Ethiopia, Kenya, Sudan, Tanzania, and Nigeria — using NASA POWER MERRA-2 satellite-derived reanalysis data spanning January 2015 to March 2026. The data was extracted for each country's capital city as a representative location.

The business objective is to produce a data-driven climate vulnerability analysis to inform Ethiopia's position paper for COP32 (Addis Ababa, 2027). Specifically, the analysis aims to:

  • Identify temperature and precipitation trends across the five countries
  • Detect extreme weather events (heat days, drought streaks)
  • Rank countries by climate vulnerability
  • Provide evidence-based recommendations on which country Ethiopia should champion for priority climate finance at COP32

The dataset contains 10 climate variables including mean, max, and min temperature (T2M, T2M_MAX, T2M_MIN), precipitation (PRECTOTCORR), relative humidity (RH2M), wind speed (WS2M), surface pressure (PS), and specific humidity (QV2M). Missing values are encoded as -999 (NASA's sentinel value) and must be handled before any analysis.

Task 2: Data Profiling, Cleaning & EDA

Five separate Jupyter notebooks were created, one per country, each following the same structured pipeline:

  • Data Loading: Each CSV loaded with a Country column added. Date reconstructed from YEAR and DOY columns using pd.to_datetime(df['YEAR'] * 1000 + df['DOY'], format='%Y%j'). Month extracted for seasonal analysis.
  • Missing Values: All -999 sentinel values replaced with NaN. Missing value percentages computed per column. No column exceeded 5% nulls in any country dataset.
  • Duplicates: No duplicate rows were found in any of the five datasets, consistent with clean daily NASA POWER extractions.
  • Summary Statistics: df.describe() run on all numeric columns. Key findings: Sudan has the highest mean T2M (~30°C), Ethiopia the lowest (~17°C) due to high altitude. Nigeria has the highest average precipitation.
  • Outlier Detection: Z-scores computed for T2M, T2M_MAX, T2M_MIN, PRECTOTCORR, RH2M, WS2M, WS2M_MAX. Outliers (|Z| > 3) were retained as they represent genuine extreme weather events critical for vulnerability analysis.
  • Cleaning: Missing values forward-filled using ffill(). Rows with >30% missing values dropped. Cleaned CSVs exported to data/ folder (excluded from GitHub).
  • Visualizations produced: Monthly T2M line charts, monthly PRECTOTCORR bar charts, correlation heatmaps, scatter plots (T2M vs RH2M; T2M_RANGE vs WS2M), log-scale precipitation histograms, and bubble charts.

Task 3: Cross-Country Comparison (Completed)

A compare_countries.ipynb notebook was created loading all five cleaned datasets. Key outputs include:

  • Monthly T2M line chart for all 5 countries on a single plot
  • Temperature and precipitation summary tables (mean, median, std)
  • Side-by-side precipitation boxplots
  • Extreme heat days per year (T2M_MAX > 35°C) — Sudan leads significantly
  • Max consecutive dry days per year — Sudan again leads
  • One-way ANOVA confirming statistically significant temperature differences (p < 0.05)
  • Vulnerability ranking table with normalized scores

Key finding: Sudan ranks highest on all vulnerability metrics — highest temperatures, most extreme heat days, longest dry streaks, and most erratic precipitation. Ethiopia ranks in the middle, with altitude moderating temperature but high sensitivity to monsoon variability.
